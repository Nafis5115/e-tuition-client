📚 Project Overview – eTuitionBd

eTuitionBd is a full-stack Tuition Management System designed to connect students and tutors in a seamless and efficient way. The platform allows students to post tuition requirements, tutors to apply for suitable opportunities, and administrators to monitor and manage the entire system.

This project aims to solve the real-world problem of finding qualified tutors while ensuring transparency, structured communication, and secure financial transactions.

⸻

🎯 Purpose of the Project

The goal of this platform is to:
	•	Simplify the process of finding and hiring qualified tutors
	•	Reduce communication gaps between students and tutors
	•	Provide a structured workflow for tuition posting and application
	•	Ensure secure and transparent payment handling
	•	Enable admins to monitor, verify, and control platform activities

⸻

⚙️ How the System Works
	1.	Students register and post tuition requirements (subject, class, budget, location, schedule, etc.)
	2.	Tutors browse available tuitions and apply to relevant posts
	3.	Students review applications and accept tutors through secure payment
	4.	Admins manage users, approve/reject tuition posts, and monitor transactions
	5.	Platform ensures smooth communication, payment tracking, and workflow automation

⸻

🚀 Key Features

👨‍🎓 Student Features
	•	Create, update, and delete tuition posts
	•	View and manage tutor applications
	•	Accept tutors via Stripe payment
	•	Track payment history
	•	Manage profile

👨‍🏫 Tutor Features
	•	Browse and apply to tuition posts
	•	Track application status
	•	View ongoing tuitions
	•	Monitor earnings and revenue history

🛠️ Admin Features
	•	Manage users (update, delete, change roles)
	•	Approve or reject tuition posts
	•	Monitor platform transactions
	•	View analytics and revenue reports

⸻

🔐 Authentication & Security
	•	Firebase Authentication (Email & Google login)
	•	JWT-based authorization
	•	Role-based access control (Student, Tutor, Admin)
	•	Protected routes with token verification

⸻

💳 Payment System
	•	Integrated Stripe payment gateway
	•	Students pay tutors securely
	•	Platform automatically calculates:
	•	Tutor earnings (90%)
	•	Admin commission (10%)

⸻

📊 Dashboard & Analytics
	•	Role-based dashboards (Student, Tutor, Admin)
	•	Real-time data visualization (charts & graphs)
	•	Monthly revenue tracking
	•	Application and tuition statistics

⸻

🎨 UI/UX Highlights
	•	Fully responsive (Mobile, Tablet, Desktop)
	•	Clean and modern UI using Tailwind CSS
	•	Smooth animations with Framer Motion
	•	Dashboard with sidebar navigation
	•	User-friendly forms and workflows
